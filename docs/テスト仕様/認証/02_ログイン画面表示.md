# 処理: GET /login（ログイン画面表示）

ログイン画面を表示する。

## エンドポイント

| 項目 | 内容 |
|------|------|
| メソッド | GET |
| パス | /login |
| ルート名 | login |

## テスト項目

| No | テスト項目 | 前提条件 | 操作 | 期待結果 | 優先度 |
|----|-----------|----------|------|----------|--------|
| AUTH-LOGIN-001 | ログイン画面が表示される | - | GET /login | 200、ログイン画面がレンダリングされる | 高 |
| AUTH-LOGIN-002 | ログイン済みでログイン画面アクセス | ログイン済み | GET /login | ダッシュボードまたはホームへリダイレクト | 中 |

## 備考

- Jetstream/Fortify のログインルート
