# 処理: GET /login-via-email/{user}（ワンクリックログイン）

メール内リンクからの署名付きワンクリックログイン。有効期限24時間・1回限り。

## エンドポイント

| 項目 | 内容 |
|------|------|
| メソッド | GET |
| パス | /login-via-email/{user} |
| ルート名 | login-via-email |
| ミドルウェア | signed（署名検証） |

## テスト項目

| No | テスト項目 | 前提条件 | 操作 | 期待結果 | 優先度 |
|----|-----------|----------|------|----------|--------|
| AUTH-005 | 有効署名でワンクリックログイン | 署名付きURL、24時間以内 | GET /login-via-email/{user}（有効署名） | ログイン成功 | 中 |
| AUTH-006 | 無効署名でワンクリックログイン | - | GET /login-via-email/{user}（無効・改ざん署名） | 403 エラー | 中 |
| AUTH-LINK-003 | 署名なしでアクセス | - | GET（署名クエリなし） | 403 エラー | 中 |
| AUTH-LINK-004 | 期限切れ署名でアクセス | 24時間経過した署名 | GET | 403 エラー | 中 |

## 備考

- `signed` ミドルウェアにより、URL に署名がない、改ざんされている、期限切れの場合は 403
- 使用コントローラ: `App\Http\Controllers\Auth\LoginViaEmailController`
