# 処理: GET /（トップアクセス）

認証状態に応じて適切なページへリダイレクトする。

## エンドポイント

| 項目 | 内容 |
|------|------|
| メソッド | GET |
| パス | / |
| ルート名 | - |

## テスト項目

| No | テスト項目 | 前提条件 | 操作 | 期待結果 | 優先度 |
|----|-----------|----------|------|----------|--------|
| AUTH-001 | 未ログインでトップアクセス | - | GET / | ログイン画面へリダイレクト | 高 |
| AUTH-002 | ログイン済みでトップアクセス | ログイン済み | GET / | /home へリダイレクト | 高 |

## 備考

- リダイレクト先: 未ログイン → `route('login')`、ログイン済み → `route('home')`
